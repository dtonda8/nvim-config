# nvim-config

### Setup (assuming no existing nvim config)
Plugins:
```bash
git clone git@github.com:dtonda8/nvim-config.git $HOME/.config/nvim-plugins
```

Config (i.e. this repo):
```bash
git clone git@github.com:dtonda8/nvim-config.git $HOME/.config/nvim
```
